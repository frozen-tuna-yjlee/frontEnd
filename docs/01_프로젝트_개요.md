# 프로젝트 개요

## 1. 프로젝트 정보

### 프로젝트명
**Portal - React 포털 사이트**

### 프로젝트 설명
한국형 포털 사이트를 React와 TypeScript로 구현한 프론트엔드 프로젝트입니다. 뉴스, 서비스, 연예/스포츠 정보를 제공하며, 카카오 소셜 로그인을 지원하는 통합 포털 플랫폼입니다.

### 버전
- **현재 버전**: 0.0.0
- **개발 브랜치**: develop

---

## 2. 기술 스택

### 핵심 기술
- **React 18.2.0** - UI 라이브러리
- **TypeScript 5.4.0** - 타입 안전성
- **Vite 5.2.0** - 빌드 도구 및 개발 서버

### 스타일링
- **Tailwind CSS 4.1.11** - 유틸리티 기반 CSS 프레임워크
- **PostCSS 8.5.6** - CSS 후처리
- **Autoprefixer 10.4.21** - 브라우저 호환성

### HTTP 클라이언트
- **Axios 1.12.2** - HTTP 요청 라이브러리

### 인증
- **Kakao SDK** - 카카오 소셜 로그인
- **js-cookie 3.0.5** - 쿠키 관리

### 아이콘
- **Font Awesome 6.4.0** - 아이콘 라이브러리

### 개발 도구
- **ESLint 9.6.0** - 코드 린팅
- **TypeScript ESLint 8.2.0** - TypeScript 린팅

## 7. 프로젝트 특징

### 7.1 타입 안전성
- TypeScript를 사용하여 컴파일 타임에 타입 체크
- 인터페이스와 타입 정의로 안전한 코드 작성

### 7.2 컴포넌트 기반 아키텍처
- 재사용 가능한 컴포넌트 구조
- 관심사 분리 (Layout, Section, Widget, Card)

### 7.3 상태 관리
- React Context API를 사용한 전역 상태 관리
- AuthContext를 통한 인증 상태 관리

### 7.4 HTTP 요청 관리
- Axios 인스턴스를 통한 중앙화된 API 관리
- 요청/응답 인터셉터를 통한 자동 토큰 처리
- 에러 처리 및 리다이렉트 자동화
